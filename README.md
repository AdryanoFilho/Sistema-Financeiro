# Sistema Financeiro API

API REST desenvolvida com Spring Boot para gerenciamento financeiro.

## Funcionalidades

- Cadastro de Categorias
- Cadastro de Pessoas
- Cadastro de Receitas e Despesas
- Atualização de Registros
- Exclusão de Registros
- Paginação e Ordenação

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Flyway
- Maven

## Estrutura do Projeto

src/main/java
src/main/resources

## Como Executar

1. Configurar o MySQL
2. Ajustar application.properties
3. Executar:

mvn spring-boot:run
